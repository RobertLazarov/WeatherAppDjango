# WeatherAppDjango

A Django-based weather app project from the Python Developer Academy.

This repository contains the source for a simple weather application built with Django. Search a city and view its current weather (temperature °C, humidity, pressure, description, icon). Successful lookups are stored and the last 5 recent searches are shown.

City search (OpenWeather “Current Weather” API)
Weather card with icon, temp, humidity, pressure, description
Recent searches list (last 5, most recent first)
Friendly errors (missing key, city not found, network)
Basic tests (smoke + view tests)
